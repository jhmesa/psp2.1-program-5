Instrucciones para descargar el codigo fuente desde GitHub: git clone https://github.com/jhmesa/psp2.1-program-5.git
Instrucciones para probar el programa en heroku: Ingrese en el navegador la url https://program5.herokuapp.com/calculation?xFrom=<valor>&xTo=<valor>&dof=<valor>
												Ejemplo: https://program5.herokuapp.com/calculation?xFrom=0&xTo=1.1&dof=9
Instrucciones para compilar el codigo fuente: Ejecute el comando mvn package desde psp2.1-program-5
Instrucciones para desplegar la aplicación localmente:  Ejecute el comando heroku local web desde psp2.1-program-5
Instrucciones para ejecutar el programa: Ingrese en el navegador la url localhost:5000/calculation?xFrom=<valor>&xTo=<valor>&dof=<valor>
										 Ejemplo: localhost:5000/calculation?xFrom=0&xTo=1.1&dof=9
